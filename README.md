# Dotfiles

## Install

### Linux

As `root`:

```bash
# apt-get install git
# git clone https://github.com/tbalthazar/dotfiles.git /tmp/dotfiles
# cd /tmp/dotfiles
# bin/install setup
# reboot
```

As `tb`:

```bash
$ git clone https://github.com/tbalthazar/dotfiles.git ~/dotfiles
$ cd dotfiles
$ bin/install dotfiles
$ sudo bin/install docker
```
